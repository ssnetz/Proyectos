# Sistema de Control de Combustible

Municipalidad de Cosquín — Control de carga de combustible para flota vehicular.

## Módulos previstos
- Vehículos (patente, tipo, área responsable)
- Conductores
- Cargas (fecha, vehículo, conductor, litros, km)
- Reportes por vehículo / área / período

## Stack
- Frontend: React + Vite
- Backend: PHP + MariaDB (XAMPP)
- Auth: sesión PHP
