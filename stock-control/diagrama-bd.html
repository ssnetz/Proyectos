Tipos de movimiento (stock_movements.type): entrada — ingreso de stock salida — egreso de stock ajuste — corrección de inventario dispensa — entrega a paciente (beneficiary_id → personas)

Las dispensas NO tienen tabla propia — se almacenan en stock_movements con type = 'dispensa' y se agrupan por reference para representar una entrega completa con múltiples ítems.
